# Solomon Ashagre - Full Stack Portfolio

**Live Demo:** [solomon-ashagre-portfolio.netlify.app](https://solomon-ashagre-portfolio.netlify.app)

## 🛠️ Tech Stack

- **Frontend:** React, Vite, CSS3
- **Backend:** Node.js, Express, MongoDB
- **Deployment:** Netlify (Frontend), Render (Backend)

## ✨ Features

- Responsive design with dark theme
- Project showcase with filtering
- Contact form (saves to MongoDB)
- PDF resume download
- Mobile-friendly navigation

## 🚀 Quick Start

# Clone repository
git clone https://github.com/Sol-Ethio-Coder/portfolio-website.git

# Install backend
cd server && npm install

# Install frontend
cd ../client && npm install

# Run (two terminals)
npm run dev  # in both server and client
